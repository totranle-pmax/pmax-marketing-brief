⚙️ Hướng dẫn kết nối Google Sheets + GitHub Pages ✕
Bước 1 — Tạo Google Sheet
Vào sheets.google.com → tạo spreadsheet mới. Đặt tên: PMAX Marketing Briefs. Sheet "Briefs" sẽ được Apps Script tự tạo với header đúng chuẩn.
Bước 2 — Mở Apps Script
Trong Google Sheet: Extensions → Apps Script. Xóa toàn bộ code mặc định trong file Code.gs.
Bước 3 — Dán code backend
Copy toàn bộ nội dung file Code.gs đã được cung cấp. Dán vào editor → nhấn Save (Ctrl+S).
Bước 4 — Deploy Web App
Nhấn Deploy → New deployment. Type: Web app | Execute as: Me | Who has access: Anyone. Nhấn Deploy → copy URL dạng: https://script.google.com/macros/s/.../exec
Bước 5 — Kết nối với form web
Mở trang index.html → dán URL vào banner xanh → nhấn Lưu. Từ đây mọi brief gửi đi sẽ ghi thẳng vào tab "Briefs".
Bước 6 — Deploy lên GitHub Pages
Tạo repo mới trên GitHub (ví dụ: pmax-brief). Upload file index.html. Settings → Pages → Branch: main → Save. GitHub cấp link: https://yourname.github.io/pmax-brief — share cho toàn công ty.
